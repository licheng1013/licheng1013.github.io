                                      # z-驾照考试

                                      • 2021/10/3

                                      # 科目3考试要点✋

                                      • 1：注意后方来车，起步左转向灯，按两下喇叭

                                      • 2：挂挡位，一档到三挡先退回空挡，一档or二挡互换挡无需回空挡

                                      • 3：座位，怎么舒服怎么来

                                      • 4：第一根调高低，第二个调前后座位，滚盘调后移

                                      • 5：打转向灯至少持续3秒，下左上右 1挡20公里以内，2挡20-30公里以内回1档速度要降到15-20以内，3挡低于25-40公里以内。其他全挂，10秒内要到规定档位速度

                                      • 6：任何挡都要松油门踩离合挂挡，一定松完离合放其他地方，踩加速加油，

                                      # 规则🗾

                                      提示

                                      注意:降速刹车要轻踩

                                      • 重要起步: 左转灯，按喇叭两下，挂挡，松手刹，松离合抖动，松刹车。

                                      • 重要:

                                        1，离合只需要换挡时候使用
2，刹车无需和离合一起使用(起步需要)
3，减速先松油门在轻踩刹车

                                      # 灯光😆

                                      • 1：夜间会车，(近光灯)
                                      • 2：无信号灯，弯道，超车，(远近交替灯)
                                      • 3：临时停车，(示廓灯)
                                      • 4：照明不好，(远光灯)

                                      # 考试必记🪲

                                      • 1：必须挂两次3档，持续开5秒以上
                                      • 2：车速如果从3档掉太快。请直接挂1档
                                      • 3：减档离合要慢抬，加档离合可以快抬
                                      • 4：路口停车必须踩离合和刹车
                                      • 5：起步只能一档起步谨记，且必须半联动

                                      # 广汕路考试路况😎

                                      提示

                                      开始考试: 逆时针巡视车辆>灯光考试>调整座椅>系好安全带>开车出去

                                      • 1：考试路口出来，右灯，肩膀到蓝色柱子停车，
                                      • 2：出来后进黄色车道，左灯换道，然后在换道
                                      • 3：出来一个掉头，4/1加一点，第二个掉头，打一圈。
                                      • 4：回去到考试出来地方，压到黄线停车，压到井盖停车右灯右打一圈，进去后一直左灯

                                      # 现场考试步骤(重要)🗾

                                      重要

                                      • 1：坐车上不要系安全带，刷身份证，开门逆时针走一遍车，上车
                                      • 2：上车，调整座位，观察灯光是否归位，系安全带，灯光模拟
                                      • 3：起步左灯，两下喇叭，挂1挡，松手刹，挂右灯3秒以上，追线30厘米以内。完成后停车
                                      • 4：起步左灯，两下喇叭，挂1挡，松手刹，开出去
                                      • 5：听到“会车”语音，打“右转向灯”，开到“蓝色柱子”停车，注意观察，上坡感觉车头往下面压“右打一圈”，挂左灯一档前面走切换“车道”
                                      • 6：听到“切换车道语音”，“左灯继续换道”，直走,如果路况良好。请加油换2-3挡。
                                      • 7：听到“通过路口”点一下“刹车”过3-5米后在点刹车，前方需要换道，打右灯。
                                      • 8：偶尔瞄一下右边牌子，”学校考试车牌“两字，肩膀过了点一下”刹车“过3-5米后在点刹车
                                      • 9：打前方路况如何：没车得情况下，可以适当换挡。上坡前需要打左灯
                                      • 10: 前方掉头，4/1 多加一点，掉头完毕。看车况换挡
                                      • 11：下坡踩刹车降低速度换1挡速度必须慢。离合必须慢放。注意前方”公交考试车牌“ 肩膀过了点一下”刹车“过3-5米后在点刹车
                                      • 12: 此时应该已经到了。超车项目，速度必须慢下来，听到请立即超车才进行超车。前面会播报一句进入超车项目
                                      • 13：挂左灯，观察路况适当给点油门切换车道过去。听到返回原车道打右灯，观察路况适当给点油门切换车道过去
                                      • 14：直线行驶，一档过去。听到直线行驶结束准备打左灯切换车道
                                      • 15：掉头行驶，左一圈+一点
                                      • 16：回到第二个车道，打右灯，行驶到加油站那边去。此时路况良好得话，请换挡加速，快到驶出路口前把速度降到1挡行驶
                                      • 17：压倒黄线即可停车(速度需要很慢)，压到井盖一点，停车。打方向盘一圈。下坡时轻点刹车一下。
                                      • 18：进去一段距离后，打左灯一直扶着。因为前方会转弯
                                      • 19：进去后肩膀对齐柱子停车。在继续启动压着盖子停车左打一圈进去。
                                      • 20：考试结束！

                                      警告

                                      • 1：踩离合加油挂
                                      • 2：没点刹*挂
                                      • 3：忘记打灯挂
