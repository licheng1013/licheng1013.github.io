    # Function

    • 1: Supports entity generation for linked database tables of the gorm framework
    • 2: Support interface and entity generation for the linkage database of gorm+gin framework
    • 3: Support generating json tag and form tag
    • 4: Supports automatic mapping to linkage attributes in the gorm framework based on fields or comments, and automatically creates and modifies time
    • 5: Support linkage optimistic lock based on field description
